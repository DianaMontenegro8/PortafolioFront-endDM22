Umas das Tecnologias usadas no meus Projectos

ReactJS

É uma biblioteca Javascript de código aberto projetada para criar interfaces de usuário com o objetivo de facilitar o desenvolvimento de aplicativos em uma única página.

Ventajas

  • Torna o processo geral de escrita de componentes mais fácil.
  • Aumenta a produtividade e facilita o aumento da manutenção
  • Garante uma renderização mais rápida

Desventajas

  • Ausência de documentação oficial.
  • Requer muito tempo para dominar, bem como um profundo entendimento de como integrar interfaces de usuário em estruturas MVC.
Javascript

Qualquer página da Web na Internet é construída, no mínimo, por HTML ( uma linguagem de marcação)e CSS(uma linguagem de estilo)

Ventajas

  • Velocidade - JavaScript tende a ser muito rápido porque muitas vezes é executado imediatamente no navegador. .
  • Simplicidade - A sintaxe JavaScript é inspirada em Java e é relativamente fácil de aprender em comparação com outras linguagens de programação populares como C++.

Desventajas

  • Ausência de documentação oficial.
